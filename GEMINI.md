# TSGabrielle (Peach Phoenix LLC) Main Storefront Context

Welcome to the TSGabrielle Next.js Storefront Project repository. This document serves as the top-level index to provide instructions and context about this project.

## Project Structure & Architecture

This is a modern full-stack web application with the following key technologies:
- **Framework**: Next.js 14+ (App Router) located in `src/app/`
- **Styling**: Tailwind CSS, PostCSS, Framer Motion
- **Database**: PostgreSQL with Prisma ORM (`prisma/` directory)
- **Deployment Structure**: Google Cloud Run targeting `tsgabrielle.com`
- **Testing**: Playwright for E2E testing (`e2e/` directory)
- **Source of Original Assets**: `stitch_tsgabrielle_home_page/` contains exact vanilla HTML/CSS reference code and mockups generated by UI Web Modern / Stitch.

## Key Directories

- `src/`: contains all React components (`src/components`), hooks (`src/lib`), Layouts, and standard Next.js directory structure inside `app/`.
- `prisma/`: contains the schema definitions, migrations, and database scripts.
- `stitch_tsgabrielle_home_page/`: Always refer to this directory when performing "Frontend Visual UI" work. It holds the source of truth for visuals.
- `e2e/`: Contains end-to-end tests covering user flows and site behavior.
- `.agent/skills/`: Holds numerous standard operating procedures and skill documents. 

## Development Conventions

1. **Aesthetics & UI**: Use extreme precision to match the visual reference files when converting raw HTML/CSS from `stitch_tsgabrielle_home_page` into modular React/Tailwind code inside `src/`.
2. **Quality Assurance**: After performing deployments or large feature upgrades, run the prescribed tests to ensure no regressions occur.
3. **Deployment Process**: Follow the steps in `.agent/skills/PeachPhoenixDeployment/SKILL.md` precisely whenever deploying to production.
4. **Environment Variables**: Managed via securely loaded processes. If the database crashes, ensure the `DATABASE_URL` matches the production Postgres credentials.

Refer to the individual `GEMINI.md` specific to `stitch_tsgabrielle_home_page` for a catalog of the existing raw UI component projects.
