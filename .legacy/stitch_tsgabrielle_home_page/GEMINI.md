## How to Work with This Project

When asked to make changes to the storefront UI or implement a new page:
1. Refer to the corresponding `code.html` to understand the exact structure, animations, and typography required.
2. The `screen.png` should be used as a visual reference for the desired appearance.
3. Extract necessary UI styles (Tailwind or raw CSS) when converting these designs to React components for the Next.js `src/` directory. Target a 1:1 visual match.
